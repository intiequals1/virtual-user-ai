# Continuous Documentation Entry 13 — Business Model

## Summary
A first business model was added for Virtual User AI.

## Main business position
The product should start as a disclosed, controlled AI meeting assistant rather than an autonomous digital human.

## Core revenue logic
- SaaS tiers
- enterprise controls and governance
- onboarding and implementation packages
- domain-specific packs
- managed deployment and compliance-oriented offerings

## Main strategic conclusion
The near-term priority is to turn the current POC and repository into a credible pilot-ready product baseline before aggressive monetization expansion.
