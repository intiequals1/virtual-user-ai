# Continuous Documentation Entry 14 — Self-Hosting Decision

## Summary
The product direction was refined toward a self-hosting-capable model.

## Main decision
Virtual User AI should be hostable on a local machine, private server, or cloud instance controlled by the user or customer.

## Strategic implication
This changes the preferred business model from pure SaaS-first to hybrid:
- self-hostable core
- optional managed cloud offering
- monetization through enterprise features, support, deployment, and governance

## Technical implication
Future implementation should preserve deployment flexibility and avoid unnecessary provider lock-in.
